# Privacy Policy for Epscan Chrome Extensions
**Effective Date:** 2026-06-25

This Privacy Policy explains how Epscan Industries Ltd. ("we," "us," or "our") handles data when you use any Epscan Internal Chrome Extension (the "Extension"). 

### 1. Information Collection and Data Usage
The Extension functions by reading Document Object Model (DOM) values from your active browser tab limited to specific internal company web app URLs.
* **Data Accessed:** The Extension reads specific text, HTML fields, or structural identifiers (DOM values) on your active tab.
* **Purpose:** This data is accessed solely to fulfill the core functionality of the extension, which is to extract and process information between various internal company web app tools.
* **Personally Identifiable Information:** This Extension NEVER collects, stores, or transmits any data that can be used to personally identify you WITH EXCEPTION TO your company login name within the internal company web apps.

### 2. Local Processing and Data Transmission
We value your privacy and practice data minimization:
* **Local Processing:** By default, all DOM parsing and data analysis occur locally within your browser environment. 
* **Data Transmission:** Extracted DOM data is securely transmitted via encrypted HTTPS connection to an internal endpoint at `https://epscancloud.com`. This data is is never shared with third-party data brokers.

### 3. Limited Use Compliance
We explicitly certify that our use of information received from Chrome APIs adheres to the [Chrome Web Store User Data Policy](https://developer.chrome.com/docs/webstore/program-policies/policies), including the Limited Use requirements. We do not sell, rent, or lease user web browsing activity or page content to any third parties.

### 4. Contact
If you have any questions or security concerns regarding any extension covered under this policy, please contact the Development team at jrosario@epscan.com
